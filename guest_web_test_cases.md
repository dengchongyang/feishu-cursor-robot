# 嘉宾 Web 端功能测试用例

# 一、 设置项校验 (组织级 & 路演级)

## 组织级设置-嘉宾Web端开关
### 前置条件
- 登录上市公司组织或券商研究所后台
- 进入组织级设置页面

### 测试步骤
1. 找到【嘉宾web端】按钮，默认状态应为关闭。
2. 开启【嘉宾web端】按钮，校验增量路演的默认状态。
3. 鼠标 hover 注释图标，校验提示文字：“开启后支持嘉宾通过web端上麦、视频和演示文档”。
4. 关闭【嘉宾web端】按钮，校验增量路演的默认状态。

### 预期结果
- 开启后，该组织下增量的路演【嘉宾web端】功能默认开启。
- 关闭后，该组织下增量的路演【嘉宾web端】功能默认关闭。

## 组织级设置-嘉宾免登录开关
### 前置条件
- 【嘉宾web端】已开启

### 测试步骤
1. 校验【嘉宾免登录】勾选项，默认应为不勾选。
2. 勾选【嘉宾免登录】，校验增量路演的默认状态。
3. 鼠标 hover 注释图标，校验提示文字：“勾选后支持嘉宾web端免登录参会”。

### 预期结果
- 勾选后，该组织下增量的路演【嘉宾免登录】选项默认勾选。

## 路演级设置-独立性校验
### 前置条件
- 拥有路演编辑/新增权限

### 测试步骤
1. 在新增/编辑路演页面，修改【嘉宾web端】和【嘉宾免登录】设置。
2. 保存路演并查看。
3. 返回组织级设置页面，查看全局配置。

### 预期结果
- 路演级修改仅对本场路演生效，不影响组织级全局设置。

# 二、 嘉宾免登录入会流程

## 免登录链接访问校验
### 前置条件
- 路演已勾选【嘉宾免登录】

### 测试步骤
1. 复制路演信息，校验是否包含“主持人/嘉宾网络参会：[免登录链接]”。
2. 使用 PC 端浏览器访问该链接。
3. 使用移动端（手机/平板）访问该链接。

### 预期结果
- PC 端正常进入入会设置页。
- 移动端提示“请通过 PC 网页端进行访问”。

## 入会设置页功能校验
### 前置条件
- 已通过 PC 端访问免登录链接

### 测试步骤
1. 校验导航栏信息：进门 logo、路演主题、路演状态、语言切换。
2. 填写【参会名】（必填，默认为空）。
3. 选择【用户身份】（嘉宾、主持人、联席主讲人、会议助理，默认嘉宾）。
4. 上传【参会头像】（默认缺省图）。
5. 调整【麦克风】、【摄像头】、【扬声器】开关及设备选择。

### 预期结果
- 各项设置均能正常操作，且状态能带入直播厅。

## 设备检测与进入会议
### 前置条件
- 已完成入会设置

### 测试步骤
1. 点击【进入会议】。
2. 若未通过麦克风检测，校验是否弹出声音输入检测弹窗。
3. 在检测弹窗中，点击【完成检测】（有声音输入 vs 无声音输入）。

### 预期结果
- 有声音输入：跳转进入嘉宾 web 端直播厅。
- 无声音输入：停留在当前页，麦克风显示异常，提示“检测到麦克风未授权...”。
- 虚拟账号自动添加至嘉宾列表，权限正确（场控、摄像头、文档演示等）。

# 三、 嘉宾 web 端直播厅核心功能

## 麦克风控制
### 前置条件
- 已进入直播厅

### 测试步骤
1. 点击麦克风 icon 进行静音/取消静音切换。
2. 远端管理员将该嘉宾静音/取消静音。
3. 开启麦克风后，以较小音量持续说话 3 秒。
4. 在通话过程中拔出麦克风或断开蓝牙。

### 预期结果
- 本地切换正常，状态同步。
- 远端控制后，本地有 toast 和语音提示。
- 音量过小时出现气泡提示：“音量较小，请提高说话音量...”。
- 设备断开时弹出提示：“麦克风被拔出或蓝牙连接断开”。

## 视频控制
### 前置条件
- 路演形式为视频路演

### 测试步骤
1. 点击摄像头 icon 开启/关闭视频。
2. 展开摄像头设置，切换设备及分辨率（360P 至 1080P）。

### 预期结果
- 开启/关闭有 toast 提示。
- 分辨率切换生效，画面清晰度变化符合预期。

## 文稿演示
### 前置条件
- 嘉宾拥有演示权限

### 测试步骤
1. 点击文稿演示 icon，查看文档列表。
2. 上传本地文档并开始演示。
3. 进行翻页操作、查看缩略图。
4. 点击【结束演示】并二次确认。

### 预期结果
- 文档上传及加载正常。
- 演示过程中视图同步，翻页流畅。
- 结束演示后，直播厅恢复正常视图。

# 四、 引导与跳转

## 引导至主直播厅
### 前置条件
- 已进入嘉宾 web 端直播厅

### 测试步骤
1. 点击常驻提示“如需进行屏幕共享...前往”。
2. 在二次确认弹窗中点击【确认】。

### 预期结果
- 弹出二次确认弹窗。
- 点击确认后，跳转至本场路演的 PC 客户端直播厅。
